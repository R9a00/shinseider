# 基礎知識コンテンツ情報収集プロンプト

## 目的
シンセイダーの補助金基礎知識セクションを充実させるため、包括的で実用的な情報を収集し、data_input/knowledge_base/ ディレクトリに整理された形で保存する。

## 収集対象カテゴリ

### 1. 基礎知識 (foundation)
- 補助金制度の基本概念
- 申請プロセス全体の流れ
- 必要書類と準備方法
- 予算計画の立て方
- 事業計画書作成の基本

### 2. 申請実務 (application)
- 申請書の書き方のコツ
- よくある間違いと対策
- 審査のポイント
- 加点要素の活用方法
- 提出前チェックリスト

### 3. 戦略・ノウハウ (strategy)
- 採択率を上げる戦略
- 競合分析の方法
- 差別化ポイントの見つけ方
- 専門家活用のメリット
- 継続的な改善方法

## 情報収集指示

以下のプロンプトをClaude Codeで実行し、各カテゴリの基礎知識コンテンツを体系的に作成してください：

### 基礎知識カテゴリ

```bash
claude code "補助金申請の基礎知識について、以下の観点で包括的な情報を収集し、data_input/knowledge_base/foundation_basics.txt として保存してください：

## 収集する情報
1. 補助金制度とは何か（定義、目的、仕組み）
2. 補助金の種類と特徴（国、自治体、民間別）
3. 申請から採択・実行・報告までの全体フロー
4. 申請に必要な基本書類一覧
5. 予算計画の基本的な考え方
6. 事業計画書の基本構成要素

## 要求事項
- 初心者でも理解できる平易な表現を使用
- 具体例を豊富に含める
- 図表や箇条書きを活用した分かりやすい構成
- 実際の申請で役立つ実用的な情報に重点
- 最新の制度変更を反映（2024年度以降）

## 出力形式
- マークダウン形式
- 各セクションに見出しを設定
- 重要ポイントは太字で強調
- チェックリストや手順は番号付きリストで記載"
```

### 申請実務カテゴリ

```bash
claude code "補助金申請の実務的なノウハウについて、以下の観点で詳細な情報を収集し、data_input/knowledge_base/application_howto.txt として保存してください：

## 収集する情報
1. 申請書作成の具体的手順とコツ
2. 審査員の視点から見た評価ポイント
3. よくある申請ミスと防止方法
4. 加点要素の効果的な活用法
5. 提出前の最終チェック項目
6. 不採択時の分析と改善方法

## 要求事項
- 実際の申請経験に基づく実践的な内容
- 「してはいけないこと」も明確に記載
- 業種・規模別の注意点を含める
- 時間管理・スケジュール管理の重要性
- 専門用語は分かりやすく解説

## 特に重視する点
- 申請書の「伝わりやすい書き方」
- 審査基準を意識した内容構成
- 差別化を図るためのアプローチ
- 書類不備による失格の回避方法

## 出力形式
- マークダウン形式
- 手順は段階的に記載
- 注意点は別ボックスで強調
- 実例やサンプルを可能な限り含める"
```

### 戦略・ノウハウカテゴリ

```bash
claude code "補助金申請の戦略的アプローチについて、以下の観点で高度な情報を収集し、data_input/knowledge_base/strategy_advanced.txt として保存してください：

- 成功事例と失敗事例の比較

## 特に重視する点
- 申請前の情報収集と分析の重要性

- 社会的意義・政策適合性のアピール方法

## 出力形式

```

## 情報収集後の処理

上記プロンプトで収集した情報は、以下のコマンドで自動的にYAML形式に変換・統合されます：

```bash
# 収集した基礎知識を統合
claude code "data_input/knowledge_base/ の全ファイルを prompts/content_review_template.md の基準で精査し、backend/knowledge_base.yaml に統合してください。重複チェックと品質向上も実施してください"

# 最終的な品質チェック
claude code "backend/knowledge_base.yaml の内容を確認し、ユーザビリティと実用性の観点から改善提案をしてください"
```

## 期待される成果

このプロンプトにより、以下の成果物が生成されます：

1. **体系的な基礎知識**: 初心者から上級者まで対応
2. **実用的なノウハウ**: 実際の申請で即活用可能
3. **業界別ガイド**: 具体的で専門性の高い情報
4. **FAQ集**: よくある問題の迅速な解決

これらの情報により、シンセイダーのユーザーは自立的に補助金申請を進められるようになります。
